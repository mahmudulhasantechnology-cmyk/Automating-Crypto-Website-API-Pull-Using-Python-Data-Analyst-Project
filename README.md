<h2>📌 Overview</h2>
<p>
This project demonstrates how to automate <b>data extraction from a cryptocurrency API</b> using <b>Python</b> and <b>Pandas</b>.  
It builds a repeatable workflow that continuously collects, processes, and stores crypto market data for analysis.
</p>

<p>
🎯 <b>Goal:</b> Create an automated pipeline to fetch real-time cryptocurrency data and prepare it for analysis.
</p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
  <li>✔ Connect to a cryptocurrency API</li>
  <li>✔ Extract real-time data using Python</li>
  <li>✔ Clean and structure the data using Pandas</li>
  <li>✔ Automate the data collection process</li>
  <li>✔ Store data for future analysis</li>
</ul>

<hr>

<h2>🛠️ Tools & Technologies</h2>
<ul>
  <li><b>Python</b></li>
  <li><b>Pandas</b></li>
  <li><b>Requests (API handling)</b></li>
  <li><b>Jupyter Notebook / VS Code</b></li>
  <li><b>Scheduling Automation (optional)</b></li>
</ul>

<hr>

<h2>📂 Dataset</h2>
<p>
<b>Source:</b> Cryptocurrency API (e.g., CoinMarketCap / CoinGecko)<br>
<b>Description:</b> Real-time data including cryptocurrency prices, market cap, trading volume, and percentage changes.
</p>

<hr>

<h2>📊 Key Steps</h2>
<ol>
  <li><b>API Connection:</b> Sent requests to retrieve crypto data</li>
  <li><b>Data Extraction:</b> Parsed JSON response into structured format</li>
  <li><b>Data Cleaning:</b> Converted and formatted data using Pandas</li>
  <li><b>Automation:</b> Created a repeatable script to run at intervals</li>
  <li><b>Storage:</b> Saved cleaned data for analysis (CSV / database)</li>
</ol>
